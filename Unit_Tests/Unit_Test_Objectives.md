This environment is a large cross-sectional study. Running a 5-pillar ablation matrix across three algorithms and three environments (yielding hundreds of distinct training runs) means that any silent implementation bug will completely invalidate the comparative results. 

You need mathematically exact, highly isolated unit tests that run deterministically and quickly to avoid integration hell. Here is the comprehensive sanity-check checklist for your non-trivial RL ablation pillars, designed to be run as automated unit tests prior to full environment deployment. Save all testing scripts and results inside of `Unit_Tests/{pillar}` where the pillars are `[DistRLTests,KLTests,PopArtTests,RNDTests]`. Whenever possible, use the existin models in `learning_algorithms/` and set their hyper parameters for each test. For example turn off RND when testing other rewards such as munchausen reward. ### 1. Optimism in the face of Uncertainty (RND)
**Objective:** Verify the intrinsic reward drives exploration without destabilizing off-policy algorithms through replay buffer non-stationarity.
* [x] **N-Chain Discovery Test:** Deploy a deterministic $N$-state chain environment. Assert that the intrinsic reward $r_{int}$ spikes significantly higher at state $i+1$ compared to state $i$ upon first discovery.
* [x] **Visitation Decay Test:** Assert that $r_{int}$ for a specific state decays to near-zero upon repeated visitations.
* [x] **Off-Policy Dynamic Recalculation Verification (DQN/SAC):** Assert that $r_{int}$ is *not* stored in the replay buffer. Verify that during the gradient step, the sampled batch of states is passed through the RND networks to compute a fresh $r_{int}$ dynamically. This prevents the Q-networks from chasing stale, inflated intrinsic rewards while maintaining a cleanly serial execution pipeline.
* [x] **On-Policy Observation Normalization (PPO):** Verify that the running mean and variance trackers normalize state observations strictly *before* they are passed to the RND predictor network.
* [ ] **RND integration on nchain:** Verify that the Rainbow agents learn nchain faster with RND by running an integration test. Save extrinsic reward over time for each PPO,SAC,DQN with and without RND. 
  - Status: DQN fails with and without RND, SAC does slightly better with RND but plateus while no RND SAC eventually passes it up in performance.

### 2. PopArt (Adaptive Target Rescaling)
**Objective:** Verify that updating the scale and shift statistics does not alter the unnormalized value estimates, preventing policy disruption. PopArt is notoriously easy to implement incorrectly.
* [x] **Precise Output Preservation:** Initialize the value network and pass a dummy batch of states to obtain unnormalized outputs $V_{old}$. Manually update the PopArt running mean and variance statistics with an extreme arbitrary shift. Apply the required affine transformation to the final linear layer's weights and biases. Pass the same dummy batch through the network to obtain $V_{new}$. Assert that $V_{old} = V_{new}$ exactly, up to machine precision ($\approx 10^{-6}$).
* [x] **Scale Invariance:** Train the algorithm on a toy environment (like a 1-step bandit) with rewards multiplied by $10^4$. Train a second instance with rewards multiplied by $10^{-4}$. Assert that both instances converge to the optimal policy in approximately the same number of steps, demonstrating true scale invariance.

* [x] **Integration:** Verifies DQN (EVRainbowDQN), SAC (EVSAC), and PPO (StandardPPOAgent) each update PopArt sigma during training and converge on a bandit task regardless of reward scale ($10^4$ vs $10^{-4}$). Calls live `agent.update()` and confirms `output_layer.sigma` shifts. Tightened assertions verify |large_mean - small_mean| ≤ 20 (DQN), ≤ 30 (SAC). Root causes identified and isolated: SAC requires `munchausen=False`, `beta_rnd=0.0`, `entropy_coef_zero=True`, and `gamma=0.0` to isolate PopArt—Munchausen adds an absolute log-policy term (~-0.38) that swamps rewards at scale $10^{-4}$, and gamma>0 causes bootstrap mismatch between burn-in sigma and true $Q^*$. `min_std=1e-8` exposed in all three algorithm constructors so it cannot mask scale-variance. All three pass (100 seeds each, ~6 min total).

### 3. Distributional Value Estimates (IQN)
**Objective:** Verify the Implicit Quantile Network correctly maps continuous quantile fractions $\tau$ to the true underlying return distribution and properly computes the Huber quantile loss.
* [ ] **Bimodal Reward Matching:** Create a deterministic 1-step environment that returns a reward of $-10$ exactly $50\%$ of the time, and $+10$ the other $50\%$ of the time. Train the IQN critic. Query the network with $\tau = [0.1, 0.25, 0.75, 0.9]$. Assert that the lower quantiles estimate $\approx -10$ and the upper quantiles estimate $\approx +10$, with a sharp step-function transition around $\tau = 0.5$.
* [ ] **Quantile Monotonicity Check:** Pass a batch of states and sorted quantiles (e.g., $\tau_1 < \tau_2 < \tau_3$) to the network. Assert that the predicted quantile values are strictly monotonically increasing: $Z(\tau_1) \le Z(\tau_2) \le Z(\tau_3)$.

### 4. Mirror Descent / Proximal Optimization (Munchausen / KL-Clip)
**Objective:** Verify that the proximal constraints actively restrict policy updates and compute the log-policy terms correctly without numerical instability.
* [ ] **Munchausen Uniformity Test (DQN/SAC):** The Munchausen penalty adds a scaled log-policy term to the target. In an environment with uniformly zero rewards ($r = 0$ everywhere), Munchausen DQN should mathematically reduce to exact entropy-regularized DQN. Assert that the Q-values and policy distributions of both implementations diverge by no more than numerical error in this specific edge case.
* [ ] **Target vs. Online Log-Policy (DQN/SAC):** Assert that the Munchausen penalty term $\tau \ln \pi(a|s)$ is calculated using the *target* network (or a delayed copy of the policy), not the current online network undergoing the gradient step.
* [ ] **PPO KL-Divergence Bound Check:** During the PPO update epoch, track the exact KL-divergence between the old and new policy. Artificially spike the advantage estimates to force a massive update. Assert that the clip fraction activates and the empirical KL-divergence remains bounded near the target KL threshold, rather than exploding.
